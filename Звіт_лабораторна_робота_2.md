## Частина 1. Розмітка вебсторінок за допомогою HTML

### Мета роботи
Ознайомитися з основними принципами розмітки вебсторінок за допомогою HTML, навчитися правильно створювати структуру HTML-документа та використовувати семантичні теги.

### Хід роботи
1. Підготовлено робоче середовище для виконання лабораторної роботи.
2. Створено файл `index.html`.
3. Реалізовано сторінку резюме (CV) з використанням семантичних тегів:
   - `<header>`
   - `<main>`
   - `<section>`
   - `<footer>`
4. Додано структурні елементи:
   - заголовки (`h1`, `h2`);
   - абзаци (`p`);
   - списки (`ul`, `li`);
   - посилання (`a`), зокрема `mailto` та посилання на GitHub.
5. Перевірено коректність HTML5-структури.

### Фрагмент коду `index.html`
```html
<!DOCTYPE html>

  <head>

  <body>
    <header>

    </main>

    <footer>
      <h2>Контакти</h2>
      <p>Email: <a href="mailto:bogdan.bach@gmail.com">bogdan.bach@gmail.com</a></p>

        <a href="https://github.com/Bogdan1239142">github.com/Bogdan1239142</a>
```

## Частина 2. Розробка вебсторінки за допомогою React (Vite)

### Мета роботи
Ознайомитися з основами React та компонентним підходом, навчитися декомпозувати інтерфейс користувача на окремі компоненти.

### Хід роботи
1. Використано структуру проєкту React, створену через Vite.
2. У папці `src/components` створено компоненти:
   - `Header.jsx`
   - `About.jsx`
   - `Experience.jsx`
   - `Education.jsx`
   - `Skills.jsx`
   - `Footer.jsx`
3. Кожен компонент реалізовано як функціональний та наповнено змістом резюме.
4. У `App.jsx` імпортовано всі компоненти та зібрано єдину сторінку.
5. Контактні дані в React-версії синхронізовано з HTML-версією.

### Фрагмент коду `App.jsx`
```jsx
import Header from "./components/Header";
import About from "./components/About";
import Experience from "./components/Experience";
import Education from "./components/Education";
import Skills from "./components/Skills";
import Footer from "./components/Footer";

function App() {
  return (
    <>
      <Header />
      <main>
        <About />
        <Experience />
        <Education />
        <Skills />
      </main>
      <Footer />
    </>
  );
}

export default App;
```
